# Swim Analyzer

Analizador de brazada de crol usando Python, OpenCV y MediaPipe

## Modulos
-pose_extractor.py - Extrae landmarks del cuerpo desde el video
- metric_calculator.py - Calcula angulos y metricas de la brazada
- rules_engine.py - Evalua si la tecnica es correcta
- ui.py - Overlay visual en tiempo real

## Uso 
python3 main.py --video ruta/al/video.mp4
